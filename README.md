# Grupo Exportador del Campo — Sitio Web Institucional

## Stack
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- PostgreSQL + Prisma ORM
- NextAuth.js (admin)
- next-intl (ES / EN / DE)
- React Hook Form + Resend

## Comandos
```bash
npm install
npm run dev        # http://localhost:3000
npx prisma migrate dev
```

## Variables de entorno
Ver `.env.example`
