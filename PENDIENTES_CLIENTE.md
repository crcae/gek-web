# Pendientes de Confirmación / Autorización del Cliente

## 1. Integración de API de LinkedIn (Home)
El plano de estructura web solicita que la sección de noticias del Home se alimente automáticamente vía la API de LinkedIn.

**Estado actual:** PENDIENTE
**Razón:** Esta integración NO está incluida en el alcance del contrato actual. Además, la API de LinkedIn para extraer posts de una página de empresa (Company Page) requiere un proceso de aprobación estricto por parte de LinkedIn que suele demorar semanas y requiere credenciales administrativas de la cuenta de la empresa.
**Solución temporal:** Se mantiene el módulo actual de "Noticias" gestionado de forma interna desde el CMS del panel administrativo (`/admin/noticias`).
**Próximos pasos:** El cliente debe autorizar esta funcionalidad como un adicional y proporcionar los accesos administrativos de LinkedIn para iniciar el proceso de revisión de la API.
